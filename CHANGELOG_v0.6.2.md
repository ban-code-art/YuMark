# YuMark v0.6.2 更新日志

**发布日期**: 2026-06-15

## 🐛 Bug 修复

### 更新对话框 Markdown 渲染
- 修复更新对话框中更新日志显示为原始 Markdown 源代码的问题
- 现在更新内容会自动渲染为格式化的 Markdown
- 支持标题、列表、粗体等格式
- 自动适配深色/浅色主题

## 📝 说明

这是一个小版本更新，主要修复了 v0.6.1 中更新对话框的显示问题。现在当应用检测到新版本时，弹出的更新对话框中的更新内容会以美观的格式化样式显示，而不是原始的 Markdown 代码。
