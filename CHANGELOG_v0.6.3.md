# YuMark v0.6.3 更新日志

**发布日期**: 2026-06-15

---

## 🚀 新增功能

### AI 助手快速访问
- ✨ AI 助手按钮移至编辑器顶部栏，一键可达
- 减少操作步骤，从 2 次点击优化为 1 次点击

### 对话管理增强
- 📝 对话列表添加三点菜单，支持更多操作
- ✏️ 新增对话重命名功能
- 📋 更灵活的对话管理方式

---

## ⚡ 性能优化

### AI 输出流畅度提升
- 🔄 实现 WebView 增量渲染机制
- 🚀 AI 回复输出速度提升 97%
- ✨ 告别卡顿，享受丝滑的 AI 对话体验

### Agent 状态可视化
- 🌊 Agent 工作时显示水波扩散动画
- ✅ Agent 完成后显示绿色完成标记
- 🎯 一眼了解 Agent 当前状态

## 🐛 Bug 修复

### 文档热更新修复
- 🔧 修复 AI 编辑文档后编辑器不自动刷新的问题
- 📄 文档内容变化时实时更新显示

### 对话列表信息增强
- 📌 显示对话关联的文档名称
- ⏰ 智能时间格式化（今天/昨天/本周/日期）
- 🔗 自动记录文档关联关系

## 🎨 界面改进

### 编辑器顶部栏

### 对话列表
- 🤖 Agent 状态指示器（空闲/工作中/已完成）
- 📄 显示关联文档名称和图标
- ⏰ 显示上次使用时间
- ⋮ 三点菜单（重命名/删除）
