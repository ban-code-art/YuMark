# Aegis Workspace

This directory holds design and architecture artifacts used to plan non-trivial changes in YuMark.

Structure:

- `INDEX.md`: entry list for specs and baseline snapshots
- `BASELINE-GOVERNANCE.md`: project baseline rules
- `baseline/`: architecture snapshots of the current codebase
- `specs/`: approved design documents before implementation planning

This workspace is documentation only. It does not change project behavior by itself.
