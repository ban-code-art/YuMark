## 🐛 可能遇到的问题

### 问题 1：启动时没有弹出更新对话框

**排查步骤**：

1. 检查应用日志：
   ```bash
   adb logcat | grep "FileListViewModel"
   ```
   查找：`启动时检查更新失败` 或相关错误信息

2. 验证 GitHub Release：
   ```bash
   curl https://api.github.com/repos/ban-code-art/YuMark/releases/latest
   ```
   确认返回的 `tag_name` 是 `v0.6.1`

3. 检查网络连接：
   - 确保手机能访问 GitHub API
   - 尝试在浏览器打开：https://github.com/ban-code-art/YuMark/releases

**可能原因**：
- GitHub API 请求失败（网络问题）
- Release 未正确创建
- 版本号解析错误

---

### 问题 2：点击"立即更新"后下载失败

**可能原因**：
- 网络不稳定
- GitHub 下载速度慢
- APK 文件过大导致超时

**解决方法**：
- 重试下载
- 切换到 WiFi 网络
- 检查手机存储空间

---

### 问题 3：下载完成后无法安装

**可能原因**：
- APK 签名不一致（两个版本用了不同的签名密钥）
- 手机安全设置阻止安装

**解决方法**：
- 确认两个 APK 都使用 `release.keystore` 签名
- 检查签名：
  ```bash
  keytool -printcert -jarfile YuMark-v0.6.0.apk
  keytool -printcert -jarfile YuMark-v0.6.1.apk
  ```
- 在手机设置中允许"未知来源"的应用安装
