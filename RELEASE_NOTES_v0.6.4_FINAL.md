## 🎉 核心更新：AI 助手对话历史功能

本次更新对 AI 助手进行了重大重构，引入了完整的对话历史记录功能，支持多轮连续对话。

### ✨ 新增功能

#### 1. 对话历史记录 💬

**现在你可以**:
- 📜 查看完整的对话历史（用户消息 + AI 回复）
- 🔄 进行多轮连续对话，无需重复打开关闭
- 📱 自动滚动到最新消息
- ⚡ 实时看到 AI 的流式回复

**使用场景**:
```
示例 1：多轮询问
用户: "这段话有什么问题？"
AI: "有几个语法错误..."
用户: "具体哪些词有问题？"
AI: "'系指'应该是'是指'..."
用户: "还有其他问题吗？"
AI: "标点符号使用不当..."
```

#### 2. 智能模式锁定 🔒

**保护机制**:
- ✅ 发送消息前：可以自由切换"询问"和"处理"模式
- 🔒 发送消息后：模式自动锁定，防止混合使用导致混乱
- 🎯 新对话时：模式重置，可以重新选择

**为什么需要锁定**:
- 避免在"询问"模式中突然切换到"处理"模式
- 保持对话上下文的一致性
- 防止误操作导致的意外结果

#### 3. 改进的消息流程 📨

**发送体验**:
- ✅ 点击发送 → 输入框立即清空
- ✅ 消息进入对话历史
- ✅ 可以立即输入下一条消息
- ✅ 错误时消息仍保留在历史中

**Agent 模式改进**:
- 🤖 底部显示"应用最新修改"按钮
- 🎯 可以多次迭代后选择最佳方案
- 📝 点击应用后自动替换原文本

## 🚀 使用指南

### 基础对话流程

1. **打开对话框**
   - 选中文本
   - 点击"✨ AI 助手"按钮

2. **选择模式**（首次可选）
   - 💬 询问：向 AI 提问
   - 🤖 处理：让 Agent 修改文本

3. **发送消息**
   - 输入内容
   - 点击发送图标或按 Enter

4. **查看回复**
   - 对话历史中显示完整记录
   - 流式查看 AI 回复

5. **继续对话**
   - 直接输入下一条消息
   - 支持多轮连续提问

### 进阶使用技巧

#### 技巧 1：渐进式提问

```
第 1 轮: "这段话有什么问题？"
（AI 给出总体评价）

第 2 轮: "能详细说说语法问题吗？"
（AI 展开说明）

第 3 轮: "那应该怎么修改？"
（AI 给出建议）
```

#### 技巧 2：Agent 迭代优化

```
Agent 模式：
第 1 轮: "改写成更专业的表达"
（查看修改版本 v1）

第 2 轮: "再简洁一些"
（查看修改版本 v2）

第 3 轮: "保持简洁但增加一些细节"
（查看修改版本 v3）

最后：点击"应用最新修改"
```

#### 技巧 3：混合使用

```
场景：不确定如何修改
1. 先用"询问"模式问 AI："这段话应该如何改进？"
2. AI 给出建议
3. 关闭对话框，重新打开（重置模式）
4. 切换到"处理"模式
5. 输入："按照刚才的建议修改"
```

## 🔧 技术改进

### 架构优化

- **对话管理**: 引入 `ConversationMessage` 数据结构
- **状态隔离**: 不同选中文本自动隔离对话
- **流式优化**: 优化流式回复的状态更新逻辑
- **内存管理**: 使用 `LazyColumn` 支持长对话

### 代码统计

```
修改文件: AiQuickDialog.kt
代码行数: 441 → 560 行
新增功能: 对话历史 (119 行)
```

## ⚠️ 已知限制

1. **对话持久化**: 对话历史仅保存在内存中，关闭应用后清空
2. **消息操作**: 暂不支持复制、编辑或删除单条消息
3. **模式限制**: 一旦发送消息，模式将锁定直到重新打开对话框
