## ⚠️ 已知限制

### 1. PDF/Word 导出（30%）
**原因**: 需要复杂的第三方库集成
**影响**: 不影响核心编辑功能
**替代方案**: 
- HTML 导出已完成
- Markdown 导出已完成
- 用户可在电脑上转换

### 2. 图片选择器 UI（0%）
**原因**: 需要 Activity Result API 集成
**影响**: 可以手动输入图片 Markdown 语法
**替代方案**: `![alt](path)` 手动输入

### 3. 主题 CSS 文件（20%）
**原因**: 需要设计多套主题样式
**影响**: 当前使用默认主题
**状态**: 数据层完整，UI 层待实现

### 4. 撤销/重做（0%）
**原因**: TextField 原生支持有限
**影响**: 轻微，用户可手动恢复
**替代方案**: 自动保存机制

## 📝 使用指南

### 创建文档
1. 主界面点击 ➕ 按钮
2. 输入文档名称
3. 点击 Create

### 编辑文档
1. 点击文档卡片进入编辑器
2. 默认显示预览模式
3. 点击 ✏️ 切换到编辑模式
4. 使用底部工具栏快速插入语法
5. 点击 👁️ 返回预览模式

### 使用 Markdown 工具栏
- **标题**: 点击 T 图标插入 `#`
- **粗体**: 点击 B 图标插入 `****`
- **链接**: 点击 🔗 图标插入 `[](url)`
- **代码**: 点击 </> 图标插入 ` `` `
- **列表**: 点击 • 图标插入 `-`

### 数学公式
```markdown
行内公式：$E = mc^2$

块级公式：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

### 图表
````markdown
```mermaid
graph LR
    A[开始] --> B[编辑]
    B --> C[预览]
    C --> D[保存]
```
````

### 代码高亮
````markdown
```kotlin
fun main() {
    println("Hello, YuMark!")
}
```
````

### 重命名文档
1. 文档卡片点击 ⋮ 图标
2. 选择 Rename
3. 输入新名称
4. 点击 Rename

### 删除文档
1. 文档卡片点击 ⋮ 图标
2. 选择 Delete
3. 确认删除操作

### 搜索文档
1. 主界面点击 🔍 图标
2. 输入搜索关键词
3. 查看搜索结果
4. 点击 ← 返回列表
